# SkillBridge - Installation & Quick Start Guide

## 🚀 Installation (No setup required!)

SkillBridge is a standalone web application that requires **NO installation**. Just open and use!

### Step 1: Locate Files
- Extract the SkillBridge folder to your desired location
- All files are ready to use

### Step 2: Open in Browser
Choose one of these methods:

**Method A: Direct File Opening**
1. Navigate to the SkillBridge folder
2. Right-click on `index.html`
3. Select "Open with" → Your browser
4. Done! 🎉

**Method B: Drag & Drop**
1. Open your browser
2. Drag `index.html` into the browser window
3. Press Enter
4. Done! 🎉

**Method C: Address Bar**
1. Open your browser
2. Press Ctrl+O (Windows) or Cmd+O (Mac)
3. Navigate to and select `index.html`
4. Done! 🎉

### Step 3: Start Using
The application opens on the home page. Navigate using:
- **Navigation Bar**: Click links to move between pages
- **Hamburger Menu**: On mobile/tablet for compact navigation
- **Direct Links**: 
  - Home: `index.html`
  - Courses: `skills.html`
  - Progress: `progress.html`

## 💾 Your Data

### How It's Stored
- All data stored in browser's Local Storage
- Persists between sessions
- No cloud upload
- Completely private

### What's Saved
1. **Completed Courses**
   - Course ID, name, completion date
   
2. **Quiz Scores**
   - Course ID, score, percentage, date
   
3. **Theme Preference**
   - Light or Dark mode choice

### Clear Your Data
- Go to Progress page
- Scroll to "Data Management" section
- Click "Clear All Progress"
- Confirms before deleting

## 🐛 Troubleshooting

### Issue: Pages not loading
**Solution**: 
- Make sure all files are in same folder
- Check folder structure matches
- Use latest browser version

### Issue: Dark mode not working
**Solution**:
- Click moon/sun icon again
- Refresh page
- Clear browser cache

### Issue: Search not finding courses
**Solution**:
- Try different keywords
- Search is case-insensitive
- Check spelling
- Use shorter terms

### Issue: Quiz score not saving
**Solution**:
- Complete all questions
- Check Local Storage is enabled
- Try in incognito/private mode
- Check browser storage settings

### Issue: Progress not showing
**Solution**:
- Complete course first
- Take quiz to generate score
- Go to Progress page
- Refresh page if needed
